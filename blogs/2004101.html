matlab中字符串的拼接三种方法

方法一：通过c = [a1, b1]格式拼接

a1 = 'abcd';

b1 = 'ecf';

c = [a1, b1];

输出结果为：c = 'abcdecf'

方法二：通过strcat函数进行拼接

d = strcat(a1, b1);

输出结果为：d = 'abcdecf'

方法三：通过sprintf函数进行拼接

e = sprintf('%s%s', a1, b1);

输出结果为：e = 'abcdecf'
